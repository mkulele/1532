08. 하드디스크 기본공유 제거
레지스트리값 AutoShareWks가 0이며 기본공유가 존재하지 않을 경우(IPC$ 제외) 
09. SAM 파일 접근통제
SAM파일 접근권한이 Administrator, System 그룹만 모든 권한으로 등록되어 있는 경우 
10. 최신 서비스팩 적용 
inciter mws client가 설치되어 있어 최신 서비스 팩이 설치되어 있음 
11. 공유권한 및 사용자그룹 설정 
공유디렉터리가 없거나 공유 디렉터리 접근 권환에 everyone이 없음 
12. 로그오프나 워크스테이션 잠김
화면보호기를 설정하고 암호를 사용하며 대기 시간이 5분 
13. 이벤트 뷰어 설정 
최대 로그크기 10240KB이상, 로그 덮어쓰지 않음 
14. 마지막 로그온 사용자 계정 숨김 
마지막 로그온 사용자 숨김 설정이 “사용”으로 설정되어 있을 경우 
15. 로그온하지 않은 사용자 시스템 종료 방지
로그온하지 않고 시스템 종료 허용'이 '사용안함'으로 설정되어 있을 경우 
16. 백신 프로그램 설치 
백신 프로그램이 설치되어 있음 
17. Null Session 설정 
해당 레지스트리 값이 설정되어 있음 
18. 레즈스트리 보호차단 
Remote Registry Service가 중지되어 있음 
19. AutoLogon 기능제어
autoadminlogon 값이 없거나 0으로 설정되어있음
